Prosjekt
Jeg lager en nettside for restauranten Tangen Torv. Brukere skal kunne se menyen til
restauranten og reservere bord. Ansatte skal ha en side hvor de kan se bestillinger.

Teknologier
HTML, CSS og  JavaScript (frontend)
PostgreSQL (database)
Docker (kjøre databasen)
NodeJS (backend – etter jul)
GitHub 
VSCode 

Database
Jeg bruker PostgreSQL.



Plan for arbeidet
Dag 1
Lage GitHub-repository
Opprette README.md og Plan.md
Lage enkel database-skisse

Dag 2
Lage SQL for tabeller
Teste innsetting og uthenting av data (INSERT og SELECT)

Dag 3
Lage HTML-sider:
hjem
meny
reservasjon
ansatte 
Style siden med CSS

Dag 4
Test av nettsiden
Opprydding i filer
Oppdatere dokumentasjon

Fokus
Enkelt design
Ren og lett kode
God dokumentasjon
Ikke flere funksjoner enn nødvendig

Etter jul
Koble frontend til backend
Backend snakker med PostgreSQL
Ansatte kan se bestillinger direkte fra database
